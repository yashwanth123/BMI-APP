# 💪 BMI Calculator – Flutter App

A beautifully designed **BMI Calculator App** built using **Flutter**. This app allows users to input their weight, height, gender, and age, and calculates their Body Mass Index (BMI) with categorized results and UI feedback.

---

## 📱 Features

- 🎨 Clean and intuitive UI with card-based input
- 👤 Gender toggle with visual feedback
- 📏 Adjustable sliders for height
- ➕➖ Buttons for weight and age control
- ✅ BMI result with category (Underweight, Normal, Overweight, Obese)

---

## 🛠 Built With

- Flutter SDK
- Dart
- Stateless and Stateful Widgets
- Custom Widgets (`ReusableCard`, `RoundIconButton`, etc.)
